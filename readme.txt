A comprehensive WordPress plugin for safely managing and patching other installed plugins directly from the WordPress admin interface.

== Description ==

Quick Patch Manager is a powerful WordPress plugin designed to help developers and site administrators make quick, safe modifications to installed plugins without directly editing files through FTP or file managers.

Key Features:
- Select and patch any installed WordPress plugin
- Comprehensive file editing interface with syntax highlighting
- Automatic backup of original files before patching
- Version control and patch history tracking
- Restore files to original or previous versions
- Detailed change logs and audit trails
- Restricted to administrator access for security

=== Use Cases ===
- Quick bug fixes for third-party plugins
- Temporary workarounds before official plugin updates
- Development and testing of plugin modifications
- Maintaining custom plugin patches

=== Requirements ===
- WordPress 5.6 or higher
- PHP 7.2 or higher
- Administrator-level user access

== Installation ==

1. Upload the `quick-patch-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the 'Patch Manager' menu in the WordPress admin dashboard

== Frequently Asked Questions ==

= Is it safe to use this plugin? =
Quick Patch Manager includes multiple safety features:
- Automatic file backups
- Restricted to administrator access
- Comprehensive change tracking

= Can I restore a file after patching? =
Yes, the plugin maintains a complete history of patches, allowing you to restore files to previous versions easily.

= Does this replace plugin updates? =
No, this is a temporary solution for quick fixes. Always update plugins through official channels when possible.
